<h1>Reserva de Café — Plataforma Web de Comercio Electrónico</h1>

<p><strong>Reserva de Café</strong> es un proyecto académico orientado al diseño, desarrollo e implementación de una
plataforma web de comercio electrónico para una cafetería local con proyección de distribución a nivel nacional.</p>

<hr>

<h2>Descripción del Proyecto</h2>

<p>
El proyecto <strong>Reserva de Café</strong> tiene como finalidad digitalizar el proceso de venta de una cafetería
especializada en café arábigo de altura. Antes de la implementación del sistema, el negocio operaba únicamente
de forma presencial, lo que limitaba su alcance y crecimiento.
</p>

<p>
La solución propuesta consiste en una <strong>tienda online funcional</strong> que permite a los clientes
visualizar productos, realizar pedidos y efectuar pagos en línea de manera segura,
mejorando la experiencia del usuario y fortaleciendo la presencia digital del negocio.
</p>

<hr>

<h2>Objetivo General</h2>

<p>
Desarrollar una plataforma web de comercio electrónico que permita a Reserva de Café
gestionar productos y pedidos, ofreciendo a los clientes una experiencia de compra
segura, clara y accesible.
</p>

<h2>Objetivos Específicos</h2>

<ul>
  <li>Implementar un catálogo digital de productos.</li>
  <li>Habilitar el proceso de compra mediante carrito de compras.</li>
  <li>Integrar una pasarela de pagos local segura (PayPhone).</li>
  <li>Gestionar pedidos desde un panel administrativo.</li>
  <li>Desplegar la solución en un entorno productivo estable.</li>
</ul>

<h2>Arquitectura</h2>

<p>
El sistema está basado en la arquitectura <strong>MVC</strong> proporcionada por WordPress,
utilizando WooCommerce como motor de comercio electrónico para la gestión de productos,
pedidos y pagos.
</p>
